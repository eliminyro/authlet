# authlet

A small, embeddable OAuth 2.1 / OIDC Authorization Server library for Go.

Domain-agnostic. Pluggable upstream OIDC IdP. JWT access tokens, opaque rotating refresh tokens, RFC 7591 DCR, PKCE, RFC 8707 audience binding, RFC 9728 PRM helpers for MCP.

## Status

Pre-1.0. Under construction.
